# YouTube Comment Summarizer Frontend

This is a React app for your project. The backend (FastAPI) must be running at http://localhost:8000 by default.

## Getting started

1. Install dependencies:
   ```
   npm install
   ```

2. Start the development server:
   ```
   npm start
   ```

## Structure
- `src/App.jsx` (main logic)
- `src/components/` (custom UI components)

For animation, uses Framer Motion (`npm install framer-motion`).

## To build for production
```
npm run build
```
